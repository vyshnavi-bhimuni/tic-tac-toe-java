# Tic Tac Toe (Java)

## Overview
A console-based Tic Tac Toe game developed using Java. The game allows two players to play turn by turn and determines the winner or a draw based on the game rules.

## Features
- Two-player gameplay
- Input validation
- Winner detection
- Draw detection

## Technologies Used
- Java

## How to Run
1. Compile the program:
   javac TicTacToe.java
2. Run the program:
   java TicTacToe
